# 前端验收标准

本标准适用于跑团网站项目的前端开发验收。

## 1. 功能验收

### 1.1 用户系统
- [ ] 支持主持人(GM)账号注册和登录
- [ ] 支持玩家账号注册和登录
- [ ] 登录表单包含用户名、密码字段
- [ ] 错误提示信息显示正确

### 1.2 房间系统
- [ ] GM可以创建跑团房间
- [ ] 房间列表展示所有可加入的房间
- [ ] 玩家可以加入现有房间
- [ ] 房间内显示当前参与者列表
- [ ] 最多支持8人同时在线

### 1.3 跑团功能
- [ ] 房间内支持实时消息通信
- [ ] GM具有房间管理权限（踢人、关闭房间等）
- [ ] 骰子投掷功能正常工作
- [ ] 角色卡牌信息展示完整

### 1.4 响应式设计
- [ ] 桌面端布局正常显示
- [ ] 移动端可正常访问和操作

## 2. 性能验收

### 2.1 加载性能
- [ ] 首屏加载时间不超过3秒
- [ ] 页面切换流畅无明显卡顿

### 2.2 并发支持
- [ ] 支持6-8人同时访问房间
- [ ] 多人同时操作时无明显延迟
- [ ] WebSocket连接稳定

## 3. 安全性验收

### 3.1 认证安全
- [ ] 密码在传输过程中使用加密
- [ ] Session/Cookie安全存储
- [ ] 登录超时机制正常

### 3.2 XSS防护
- [ ] 用户输入内容正确转义
- [ ] 防止脚本注入攻击

### 3.3 CSRF防护
- [ ] 表单提交包含CSRF Token
- [ ] API请求验证来源

## 4. UI/UX验收

### 4.1 视觉一致性
- [ ] 整体配色风格统一
- [ ] 字体大小和层级清晰
- [ ] 按钮样式一致

### 4.2 交互反馈
- [ ] 按钮点击有视觉反馈
- [ ] 加载状态显示Loading
- [ ] 操作成功/失败有明确提示

### 4.3 Accessibility
- [ ] 重要元素具有焦点状态
- [ ] 颜色对比度符合WCAG标准
- [ ] 关键操作支持键盘操作

## 5. 浏览器兼容性

- [ ] Chrome 最新版正常运行
- [ ] Firefox 最新版正常运行
- [ ] Edge 最新版正常运行
- [ ] Safari 最新版正常运行
