# 《四重门》新手教学

## 一、这是什么游戏？

《四重门》是一个**奇幻探索模组**，玩家扮演六位失忆的调查员，在一座死寂的文明废墟中醒来，寻找真相并试图回到原本的世界。

**核心特色：**
- 塔罗大阿卡纳牌命运系统
- 六位失忆调查员合作探索
- 跨越位面的奇幻冒险

---

## 二、创建你的调查员

### 1. 选择职业

首先，选择你想扮演的职业。职业决定了你的特殊能力。

#### 非施法职业

| 职业 | 职业专长 | 特殊能力 |
|------|----------|----------|
| 事务所侦探 | 社交×1、格斗、射击、潜行、追踪、侦查 | 所有侦查难度等级-1 |
| 运动员 | 攀爬、跳跃、格斗、社交×1、游泳、投掷 | 每场游戏一次，可重掷力量或敏捷检定 |
| 工程师 | 电气维修、机械维修、操作大型机械、科学(工程/物理)、侦查、驾驶 | 可临时组装或改装设备 |
| 法医 | 医学、说服、科学(生物/鉴证/制药)、侦查、急救 | 检查尸体/死亡现场时获得关键线索 |
| 猎人 | 射击、聆听/侦查、导航、生存、科学(生物/植物)、潜行、追踪 | 野外环境中无法被伏击 |
| 黑帮马仔 | 驾驶、格斗、射击、社交×2 | 与灰色地带人物社交自动获得优势 |
| 通灵师 | 驾驶、社交×1、历史、神秘学、科学(生物/化学) | 可与场所残留灵魂建立联系 |
| 间谍 | 乔装、射击、聆听、社交×1、妙手、潜行 | 识破伪装难度提升一级 |
| 游民 | 技艺×1、攀爬、跳跃、聆听、锁匠/妙手、导航、潜行 | 永远能在陌生环境找到安全过夜处 |

#### 施法职业

| 职业 | 施法主属性 | 职业专长 | 特殊能力 |
|------|------------|----------|----------|
| 法师 | 智力 | 施法、历史、神秘学、科学×1、社交×1、闪避 | 可学习魔法物品/战利品中的魔法，不消耗魔力 |
| 术士 | 魅力 | 施法、历史、考古、神秘学、社交×1 | 可增加魔力消耗以成倍增强效果/选中更多目标 |
| 吟游诗人 | 魅力 | 施法、表演技艺、其他任意 | 可学习任意专长作为职业专长，初始法术数量+2 |

> **注**：职业专长中的"社交×1"表示从社交类专长中选择一项；"社交×2"表示选择两项；"科学×1"表示从科学类专长中选择一项。

---

### 2. 选择专长

选好职业后，从职业提供的专长列表中选择你想学习的专长。**专长代表你在某一领域的专业训练**。

例如：选择"射击"专长后，你使用枪械时检定会获得加值；选择"医学"专长后，你进行医疗检定时会更熟练。

#### 专长与属性速查

| 属性 | 作用 | 人类极限 | 关联专长 |
|------|------|----------|----------|
| 力量 | 近战攻击、需要力量的动作 | 99 | 格斗、投掷 |
| 体质 | 生命值、持续伤害检定、回血 | 99 | - |
| 敏捷 | 需要敏捷的动作、闪避 | 99 | 射击、闪避 |
| 智力 | 决定可分配专长点数 | 99 | 医学、急救、聆听、导航、生存、历史、考古、神秘学、电气维修、机械维修、操作大型机械、自然科学 |
| 意志 | 初始SAN值、灵感检定 | 99 | 侦查、潜行、追踪、妙手、撬锁、驾驶 |
| 魅力 | 社交类动作 | 99 | 恐吓、说服、魅惑、表演技艺、乔装打扮 |

> **备注**：普通人属性是50点，20点是正常人的下限，80点是正常人的上限，99点是人类的极限。

---

### 3. 分配属性

属性反映你调查员的身体素质和心智能力。专长需要对应的属性支持才能发挥作用。

#### 基础属性（320点分配）

| 属性 | 作用 | 人类极限 |
|------|------|----------|
| 力量 | 近战攻击、需要力量的动作 | 99 |
| 体质 | 生命值、持续伤害检定、回血 | 99 |
| 敏捷 | 需要敏捷的动作、闪避 | 99 |
| 智力 | 决定可分配专长点数 | 99 |
| 意志 | 初始SAN值、灵感检定 | 99 |
| 魅力 | 社交类动作 | 99 |

#### 衍生属性

- **生命值**：10 + 体质÷4（向下取整）
- **初始SAN值**：等于意志值
- **魔力值**：意志÷5（仅施法职业）

---

### 4. 专长加点

将你的专长点数分配到已选择的专长上。

#### 专长点数

- **职业专长点数**：智力÷2（只能加到职业对应的专长上）
- **兴趣专长点数**：智力÷4（可以加到任何专长上）
- 专长最多5-6个

#### 加点限制

- 职业专长：属性 + 专长加值 ≤ 80
- 兴趣专长（非职业）：属性 + 专长加值 ≤ 70
- 兴趣专长（职业）：属性 + 总专长加值 ≤ 80
- **没有相应专长**：进行相关检定时，难度等级+1

---

### 5. 物品包系统

完成了职业、专长、属性三大要素后，给你的调查员选择物品包。每个物品包包含：**武器 × 1 + 工具 × 1 + 其他物品 × 1**

#### 物品包一览

| 物品包 | 武器 | 工具 | 其他物品（示例） |
|--------|------|------|----------|
| 战斗型 | 手枪 + 弹匣×2 | 格斗手套 / 甩棍 | 急救包 |
| 侦查型 | 战术匕首 | 多功能侦察工具箱（夜视镜、信号干扰器等） | 笔记与绘图用具 |
| 维修型 | 工具锤 | 维修工具箱 | 备用零件袋 |
| 医疗型 | 手术刀套装 | 便携医疗箱 | 法医检验手套 |
| 潜行型 | 绞杀绳 / 静音匕首 | 开锁工具套装 + 攀岩钩爪 | 黑色作战服 |
| 社交型 | 随身折叠刀 | 化妆伪装套装 | 名片与伪造证件 |
| 施法型 | 法杖 | 魔法材料包（蜡烛、符文纸、香料等） | 魔法书/卷轴 |
| 生存型 | 野外生存刀 | 野外套装（帐篷、绳索、炊具等） | 干粮袋（3天） |
| 通用型 | 甩棍 | 万能工具卡 | 背包（可额外携带一件物品） |

> **魔法物品**：可替代"其他物品"位置，初始每多携带一件承受 (1d5+5) × 释放次数 × 法术环数 的SAN值伤害

- 调查员可以创作符合自身背景和想法的魔法物品，携带有限次数的魔法
- 初始每多携带一件魔法物品，会额外承受 (1d5+5) × 释放次数 × 法术环数 的SAN值伤害
- 魔法物品可以替代物品包中的"其他物品"位置

## 三、核心机制

### 1. 时间系统

游戏中具体的时间流逝由GM根据剧情需要把握。玩家只需知道：

- 游戏中的昼夜更替会影响某些行动（如夜间潜行更有利）
- 某些事件有固定的时间节点，请留意GM的提示

> **注**：无需记录精确的时间消耗，专注探索和故事即可。

---

### 2. 检定系统

#### 何时需要检定？

普通人能完成的动作不需要检定。有难度的动作则需要检定。

#### 检定公式

**成功率先 = (属性 + 专长加值)%**

- 属性：车卡时对应此动作的属性
- 专长加值：精确指向这一动作的专长技能值
- 其他：如果能营造己方优势/敌方劣势，除了调整难度等级外，可能还有额外加减值

> **示例**：调查员小y试图从三楼精确的跳进地面上的一堆干草堆来规避摔落伤害。她的敏捷属性为70，同时具有5运动专长，那么基础成功率为75%，即掷出75及以下均成功。

#### 难度等级

在基础成功率的基础上进行判定：

| 难度 | 成功率 |
|------|--------|
| 普通成功 | 基础成功率 |
| 困难成功 | 基础÷2 |
| 极难成功 | 基础÷5 |

> **示例**：由于是大雾天气，小y看不清楚干草堆的具体位置，只能按照记忆来运动，成功率为(70+5)/2 = 38%

#### 加值/减值

由于一些其他原因（例如：能力、角色状态等），角色的检定可能会承受额外的加值/减值。

- 减值示例：昨天没睡好，运动检定额外承受1d10减值（在骰子结果上+1d10）
- 加值示例：具有精神方面特殊能力，抵抗魅惑类技能时获得额外的1d5加值（在骰子结果上-1d5）

#### 大成功与大失败

| 结果 | 判定范围 | 效果 |
|------|----------|------|
| 大成功 | 1-5点 | 获得意想不到的好结果 |
| 大失败 | 96-100点 | 碰到倒霉的事情 |

> **重要**：大成功和大失败无视任何难度等级、加减值和数值溢出。先在基础1d100上计算大成功大失败，再计算其他加减值，再计算难度系数。

> **示例**：
> - 小y掷出5点，精确降落到了干草堆正中心，获得了对站岗敌人的优势骰
> - 小y掷出98点，降落过程中失去平衡，砸到保安队长，双方都受到1d6伤害
> - 即使小y因为各种debuff理论上需要掷出≤3才能成功，掷出5依然被视为大成功
> - 即使小y敏捷90+运动专长20，掷出96时依然发生大失败

### 3. 战斗系统

当一次战斗触发时，进入战斗轮模式。

#### 战斗流程卡

```
┌─────────────────────────────────────────────────────────┐
│                    战斗流程                              │
├─────────────────────────────────────────────────────────┤
│  1. 攻击检定：1d100 ≤ (属性 + 专长加值)                 │
│        ↓ 失败                                          │
│     对方反击（伤害减半）                                │
│        ↓ 成功                                          │
│  2. 对方闪避检定：1d100 ≤ (敏捷 + 闪避专长)            │
│        ↓ 闪避成功                                      │
│     伤害 = 0                                           │
│        ↓ 闪避失败                                      │
│  3. 造成伤害：(基础伤害骰 + 属性调整值)                │
├─────────────────────────────────────────────────────────┤
│  属性调整值 = (力量/敏捷/施法主属性 - 50) ÷ 10         │
└─────────────────────────────────────────────────────────┘
```

#### 行动顺序

1. **敏捷排序**：敏捷越高的单位越先行动
2. **延迟行动**：可以选择将行动推迟到本轮末尾

#### 伤害公式

**伤害 = 基础伤害骰 + 属性调整值**

| 武器类型 | 基础伤害 | 属性调整 |
|----------|----------|----------|
| 徒手 | 1d3 | (力量-50)÷10 |
| 临时武器 | 1d4 | (力量-50)÷10 |
| 锐器 | 1d5+1 | 出血每回合+1 |
| 钝器 | 1d6 | 附加负面状态 |
| 投掷 | 1d4 | (敏捷-50)÷10 |
| 便携枪械 | 1d8+2 | (敏捷-50)÷10 |
| 军用枪械 | 1d8+4 | (敏捷-50)÷10 |

---

### 4. 法术系统

法术是一类强大的能力，通过消耗魔力引导超越凡人的力量。

#### 法术分级

| 等级 | 对标 | 魔力消耗 | 效果描述 |
|------|------|----------|----------|
| 日常级 | 戏法 | **0** | 无需魔力，类似小把戏，无需准备随时可施放 |
| 简单级 | 1环 | 2-4点 | 基础魔法，效果有限 |
| 中等级 | 2环 | 5-6点 | 中等魔法，有明显效果 |
| 困难级 | 3环 | 7-8点 | 强力魔法，需要技巧 |
| 传说级 | - | 15-25点 | 近乎神迹，效果惊人，可榨干全部魔力 |

#### 法术学习

- 初始可学习：智力÷20 个简单级法术
- 每次升级：+1个法术，可选同级或更低级
- 详细信息请查阅 [法表](四重门_法表.md)

> **警告**：魔力耗尽后施法将造成等同于溢出部分的SAN值和生命值伤害

#### 魔法物品

- 只有施法职业可以使用自身的"魔力值"进行施法
- 非施法职业只能通过魔法物品有限次数施法
- 初始每多携带一件魔法物品，会额外承受 (1d5+5) × 释放次数 × 法术等级 的SAN值伤害

#### 施法主属性

- **法师**：智力
- **术士、吟游诗人**：魅力
- 非施法职业通过魔法物品施法时，施法主属性默认为智力

---

### 5. 升级系统

在每一章结束时，GM会进行结算：

1. **所有PC**：选择获得4点生命值上限提升 或 魔力值上限提升

2. **所有PC**（选择一项）：
   - 武器类型转变
   - 武器伤害提升1d2
   - 学习一项新法术（仅施法职业）
   - 回复自己20%的生命值或SAN值或魔力值

## 五、职业详细说明

### 非施法职业

#### 事务所侦探

> "真相从不主动登门——它藏在被撕碎的账单边缘，藏在目击者闪烁其词的停顿里，藏在那扇永远留一条缝的铁门背后。"
>
> 他们的战场是街巷、账簿和人心。他们知道如何让一个人开口，更知道什么时候该拔枪。

- **职业专长**：一项社交技能、格斗、射击、潜行、追踪、侦查
- **特殊能力**：所有侦查的难度等级下降一级

#### 运动员

> "他们习惯于用身体对话。当别人绕路时他们翻越，当别人退缩时他们冲刺，当别人犹豫时他们已经跃起。"

- **职业专长**：攀爬、跳跃、格斗、一项社交技能、游泳、投掷
- **特殊能力**：每场游戏一次，可以在进行过一次力量或敏捷检定后，选择重掷该次检定。必须接受新的结果。

#### 工程师

> "他们不擅长站在最前面，但整个队伍都依赖着他们的双手。他们让沉默的机械开口说话，让瘫痪的铁兽重新站起。"

- **职业专长**：电气维修、机械维修、操作大型机械、科学（工程学、物理学）、侦查、驾驶
- **特殊能力**：可以临时组装或改装一件设备。消耗一定的材料和时间，可以让一件机械/电子设备临时获得一项额外的功能。

#### 法医

> "他们既是解剖台前的学者，也是法庭上的专家证人。手术刀切开的不只是皮肤和肌肉，还有谎言和偏见。"

- **职业专长**：医学、说服、科学（生物学、鉴证、制药）、侦查、急救
- **特殊能力**：当检查一具尸体或一处与死亡相关的现场时，可以进行一次医学/科学检定，以获得一条用常规手段无法获取的关键线索。

#### 猎人

> "他们走在队伍的最前方，脚步轻得像一片落叶。眼睛习惯了远距离的凝望，耳朵习惯了分辨异常的响动。"

- **职业专长**：射击、聆听或侦查、导航、生存、科学（生物学或植物学）、潜行、追踪
- **特殊能力**：在野外环境中无法被伏击。当团队遭遇突袭时，不会陷入措手不及的状态。

#### 黑帮马仔

> "他们太清楚人是什么样的了。一个眼神就知道对方是虚张声势还是真敢动手，一句话就听得出这单生意能做还是该撤。"

- **职业专长**：驾驶、格斗、射击、两项社交技能
- **特殊能力**：在与灰色地带的人物（黑帮、线人、街头混混等）打交道时，社交检定自动获得优势。

#### 通灵师

> "他们行走生者中间，目光却常常落在空无一人的角落。他们与亡魂交谈，不是为了猎奇，而是因为那些声音里藏着活人找不到的答案。"

- **职业专长**：驾驶、一项社交技能、历史、神秘学、科学（生物学、化学）
- **特殊能力**：可以尝试与特定场所残留的灵魂残响建立联系。进行神秘学检定时，成功则能从亡魂处获得一条与该地点相关的、亡者生前所知的线索。但每次使用此能力，必须进行一次意志检定，否则会承受1d6SAN值伤害。

#### 间谍

> "他们善于成为另一个人，更善于在成为那个人之后全身而退。他们不是最锋利的刀，但是插得最深、也最让人感觉不到疼的那一把。"

- **职业专长**：乔装、射击、聆听、一项社交技能、妙手、潜行
- **特殊能力**：进行乔装后，更容易被认为是所伪装的身份。识破伪装需要的难度等级提升一级。

#### 游民

> "他们像一只习惯了迁徙的鸟，对巢穴没有执念，但对方向有着近乎本能的嗅觉。他们能去别人去不了的地方，能打开别人打不开的锁。"

- **职业专长**：技艺（任一）、攀爬、跳跃、聆听、锁匠或妙手、导航、潜行
- **特殊能力**：永远能在一个陌生环境中找到一处安全、隐蔽、可以过夜的临时住所。

### 施法职业

#### 法师

> "汲取弥漫于宇宙的精妙魔力的法师，可以用法术生成爆发的火焰和射弧的闪电，或是施行微妙的欺瞒或强硬的精神控制。"

- **施法主属性**：智力
