# TRPG Online - 跑团网站项目

## 项目概述

这是一个支持 6-8 人同时访问的在线跑团(TRPG)网站，提供主持人和参与者分别登录的功能。

## 核心功能

| 功能 | 描述 |
|------|------|
| **用户系统** | 区分主持人(GM)和参与者(玩家)角色，分别登录 |
| **游戏房间** | 创建游戏房间，6-8人同时在线 |
| **界面控制** | GM 可控制各界面展示内容 |
| **资源管理** | GM 后台导入图片和文本资源 |
| **掷骰子系统** | GM 在后台掷所有人可见的骰子 |
| **笔记本功能** | 参与者可使用的笔记本界面 |

## 技术架构

- **前端**: 响应式 Web 界面
- **后端**: REST API + WebSocket 实时通信
- **数据库**: 关系型数据库

## 目录结构

```
├── src/

│   ├── evaluation/   # 验收标准

```

## 开发流程

1. 使用 `/start` 开始开发
2. 使用 `/checkpoint` 保存进度
3. 使用 `/resume` 恢复工作

## 验收标准

- 6-8 人同时在线流畅
- GM 和玩家权限正确隔离
- 实时消息同步
- 掷骰子结果所有人可见
